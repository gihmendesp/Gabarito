# 📘 Gabarito da Gi v2.0

Sistema de organização e acompanhamento de estudos para concursos públicos.

Este projeto foi desenvolvido como uma ferramenta pessoal para registro, resolução e análise de questões de concursos, com foco em desempenho e evolução contínua.

---

## 🚀 Funcionalidades

### 📚 Gestão de questões
- Cadastro de questões completas
- Disciplina, banca, concurso e ano
- Enunciado e resposta comentada
- Status por questão:
  - Não respondida
  - Acertada
  - Errada

### ⭐ Organização inteligente
- Marcar questões como favoritas
- Filtros por texto, status e conteúdo
- Busca por disciplina, assunto ou enunciado

### 📊 Estatísticas automáticas
- Total de questões cadastradas
- Total respondidas
- Número de acertos
- Número de erros
- Percentual de desempenho
- Barra de progresso com feedback automático

### 🌙 Interface moderna
- Modo claro e modo escuro
- Layout responsivo (funciona em celular e desktop)
- Design limpo e focado em produtividade

### 💾 Persistência de dados
- Dados salvos automaticamente no navegador (localStorage)
- Não perde informações ao atualizar a página

---

## 🧠 Objetivo do projeto

O objetivo do **Gabarito da Gi** é ajudar na preparação para concursos públicos, permitindo:

- Organizar melhor os estudos
- Identificar pontos fracos
- Acompanhar evolução real
- Criar disciplina de resolução de questões

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro (Vanilla JS)
- LocalStorage (armazenamento local do navegador)
